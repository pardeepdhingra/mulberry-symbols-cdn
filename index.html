Mulberry Symbols CDN

This repository serves as a CDN for Mulberry Symbols, providing SVG files for AAC (Augmentative and Alternative Communication) applications.

Usage

Access any symbol using the URL pattern:

https://pardeepdhingra.github.io/mulberry-symbols-cdn/[filename].svg
